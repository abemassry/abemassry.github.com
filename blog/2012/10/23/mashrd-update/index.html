Mashrd Update

Oct 23rd, 2012

Just wanted to tell you about some of the design choices for mashrd. I wanted to support the big social networks to start out with. I really wanted to have App.net but I could not find a button for them. Buffer, bufferapp.com has a button but I had trouble integrating it and I will delve deeper into the API soon. The other issue is for the popular page the network with the highest count gets the button. So that is usually Facebook or Twitter. When App.net has a button I can make a companion site with App.net as it’s own news listing site, if there is demand for it. I co-posted this post on the mashrd about page as well as here.
